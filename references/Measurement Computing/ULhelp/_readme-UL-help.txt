Windows only  
------------  
The help file in this ZIP archive is the latest version of the help file
installed with the Universal Library (UL) software in the Measurement Computing\DAQ folder.

You cannot view the information in this help file if you launch the file
directly from the zip file or across a networked drive.  To view this
help file, extract ULHelp.chm from this ZIP archive and copy it to an
appropriate location on your computer, then double-click the ULHelp.chm
file.  

To replace the version installed on your computer with the version
extracted from the ZIP archive, copy ULHelp.chm to the
Measurement Computing\DAQ folder. You can then access this information by
selecting All Programs>>Measurement Computing>>Universal Library>>UL Help
from the Start menu,  


Unblocking the CHM File  
-----------------------  
The security settings on your PC may require you to unblock the .chm file  
before you can view its contents. To unblock the .chm file, right-click
the file, and choose Properties from the shortcut menu. If the Unblock
button is visible at the bottom of the General or Security tab, click
that button and try to reopen the file.   
  
Refer to the Microsoft Knowledge Base (article ID 896358) for more  
information about the security updates that prevent you from viewing HTML  
Help files remotely.
